# TypeScript

- Microsoft에 의해서 개발 및 관리되고 있는 오픈소스 프로그래밍 언어로 JavaScript의 Superset이다.
- 타입을 추가하여 발생하는 에러를 초기 개발 과정에서 발견할 수 있으며 추가적인 오류 검사를 제공한다.
- 브라우저와 같은 자바스크립트 환경에서 실행할 수 없다. 컴파일러로서 작성한 코드를 자바스크립트로 컴파일한다.

<br>
